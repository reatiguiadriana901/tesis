# Análisis Morfométrico y Morfoestructural del Campo Volcánico Monogenético San Agustín-Isnos

**Autora:** Adriana Marcela Reatigui Mateus  
**Directores:** Dayana Schonwalder · Hugo Murcia  
**Institución:** Universidad Industrial de Santander, Bucaramanga, Colombia  

## Descripción
Análisis estadístico de 508 lineamientos estructurales mapeados con cuatro técnicas de sombreamiento sobre el Campo Volcánico Monogenético San Agustín-Isnos, Huila, Colombia.

## Requisitos
geopandas, numpy, pandas, matplotlib, scipy

## Datos
Los archivos de lineamientos no están en el repositorio por su tamaño. Colocarlos en la carpeta `datos/` para ejecutar el notebook.
